Motion

Motion explains state changes only — 150–250ms, ease-out, translations under a few pixels, no bounce. Hover a row to see the token in action.

--dur-base / 200msTabs, toggles, inline reveals
--dur-slow / 250msDrawers, dialogs, page transitions
